animate.leave: string | string[]

You can use animate.leave to animate elements as they leave the DOM. You can define leave animations using CSS classes with either transitions or keyframe animations.

When the animation completes, Angular automatically removes the animated element from the DOM.

You can use animate.leave with a single class or multiple classes. Either specify it as a simple string with spaces or a string array.

Library:

@angular/core@20.2.2
