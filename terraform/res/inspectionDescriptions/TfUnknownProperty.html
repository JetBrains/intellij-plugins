Reports unknown properties in blocks.

This inspection is available only if the Terraform or OpenTofu directory is initialized (i.e., terraform init or tofu init has been run), so that the IDE can fetch provider metadata.

Example of a problem:


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
  test = ""
}

In this example, the test property in the block is unknown.

Example of correction:


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
}
