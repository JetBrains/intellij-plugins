@relation(
    name: String?,
    map: String?,
    fields: FieldReference[]?,
    references: FieldReference[]?,
    onDelete: ReferentialAction?,
    onUpdate: ReferentialAction?)

Defines a connection between two models. Learn more.

Params:

 name

Defines the name of the relationship. In an m-n-relation, it also determines the name of the underlying relation table.

map

Defines a custom name for the foreign key in the database.

fields

A list of field references of the current model.

references

A list of field references of the model on the other side of the relation.

onDelete

Specifies the action to perform when a referenced entry in the referenced model is being deleted. Learn more.

onUpdate

Specifies the action to perform when a referenced field in the referenced model is being updated to a new value. Learn more.
