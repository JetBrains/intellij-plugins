Use StackSets to centrally manage stacks across accounts and regions.

StackSets enables you to create, update, or delete stacks across multiple

AWS CloudFormation StackSets.

In the console, stack operations display the client request token on the

more information, see Viewing Stack Data and Resources and StackEvent

    originate.

    constraint used to check the size of a web request and which parts of
    the request to check.

    Use the AWS::WAFRegional::SqlInjectionMatchSet resource to allow,
    block, or count requests that contain malicious SQL code in a specific
    part of web requests.

    Use the AWS::WAFRegional::WebACL resource to identify the web requests
    that you want to allow, block, or count.

AWS::WAFRegional::WebACLAssociation

    Use the AWS::WAFRegional::WebACLAssociation resource to associate a web
    access control group (ACL) with a resource.

AWS::WAFRegional::XssMatchSet

    Use the AWS::WAFRegional::XssMatchSet resource to specify the parts of
    web requests that you want AWS WAF to inspect for cross-site scripting
    attacks and the name of the header to inspect.
