Widely available across major browsers

Chrome 1, Chrome Android 18, Edge 12, Firefox 1, Firefox Android 4, Opera 15, Safari 1, Safari iOS 1

Baseline since 2015

The image URL. At least one of src and srcset is required for an <img> element. If srcset is specified, src is used in one of two ways:
  • as a fallback for browsers that don't support srcset.
  • if srcset uses the "x" descriptor, then src is equivalent to a source with the density descriptor 1x; that is, the image specified by src is used on low-density screens (such as typical 72 DPI or 96 DPI displays).
